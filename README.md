## 1.プロジェクト概要
ユーザは所在地域と、利用する各コンビニエンスストアを設定し、設定したコンビニの隔週の新商品を表示させるアプリ。また、気になる商品をお気に入り登録し、お気に入り登録数に応じた商品ランキングを掲載。
